# Advanced CLI Contact Management System

##  Project Overview

This project is a Command-Line Interface (CLI) based Contact Management System developed in Python. It allows users to efficiently manage contacts with features like adding, viewing, searching, filtering, updating, and deleting records. The system uses JSON for persistent data storage and ensures data consistency across sessions.

---

##  Objectives

* To build a real-world contact management system using Python
* To implement CRUD operations (Create, Read, Update, Delete)
* To practice file handling using JSON
* To improve problem-solving and CLI application development skills

---

##  Features

*  Add new contacts with validation
*  View all contacts in structured table format
*  Search contacts (case-insensitive)
*  Filter contacts by City or Company
*  Update existing contact details
*  Delete contacts
*  Auto-generated unique ID for each contact
*  Sort contacts alphabetically by name
*  Export contacts to CSV file
*  Persistent storage using JSON

---

##  Technologies Used

* Python 3
* JSON (for data storage)
* CSV (for export feature)
* Regular Expressions (for validation)
