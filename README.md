# 📇 CLI Contact Management System

A professional, feature-rich **Command-Line CRM tool** built in Python. Manage contacts with search, filters, sorting, pagination, import/export, and starred favourites — all from the terminal.

---

## 🚀 Features

| Feature | Details |
|---|---|
| ➕ Add Contact | Full validation (email, phone, empty fields) |
| 📋 View All | Paginated table (10/page), sortable by name/city/company/date |
| 🔍 Search | By name, phone, or email (case-insensitive partial match) |
| 🔎 Filter | By city and/or company |
| ✏️ Update | Edit individual fields by Contact ID |
| 🗑️ Delete | Remove contact by ID with confirmation prompt |
| 💾 Persistence | Auto-save/load from `data/contacts.json` |
